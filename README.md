# Simple Gui for Reciprocal Best Hit with Blast for MacOsX

Usage:
 - Download Repository
 - Open gui_rbbh.sh and insert the absolute path to the gui_rbbh.py file
 - Make gui_rbbh.sh executable (chmod +x gui_rbbh.sh)
 - Double Click and Run !
